A Java regular expression to match against the username (slug) of comment authors. Comments from users whose username matches this regex will be ignored and will not trigger builds. This is useful for excluding comments from bots or service accounts.

Leave empty to allow comments from all users to trigger builds.

Examples:

  • bot-.* — ignores comments from any user whose name starts with "bot-"
  • jenkins|bamboo — ignores comments from users named "jenkins" or "bamboo"
  • .*-bot — ignores comments from any user whose name ends with "-bot"
